# 🔐 Pinterest-Inspired Login Page

## 📌 Project Description

This project is a simple and responsive login page inspired by the design style of Pinterest. It is built using basic HTML and CSS, making it ideal for beginners who want to understand layout design, form creation, and user interface styling.

The main goal of this project is to practice front-end development fundamentals.

---

## ✨ Features

* Clean and minimal UI inspired by Pinterest
* Centered login box using Flexbox
* Email and password input fields
* Styled login button with hover effect
* Beginner-friendly code structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Internal CSS)

---

## 🎯 Learning Outcomes

Through this project, I learned:

* How to center elements using Flexbox
* Difference between justify-content and align-items
* Basic form design and styling
* Creating a visually appealing UI layout

---

## 🚀 Future Improvements

* Add form validation using JavaScript
* Implement show/hide password feature
* Connect with backend for real authentication
* Improve responsiveness for mobile devices

---

## 📷 Inspiration

The design of this login page is inspired by Pinterest's clean and modern user interface.

---

## 📎 Conclusion

This project is a beginner-level implementation that helped me strengthen my basics in web development and understand how real-world login pages are structured.
